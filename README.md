#### &nbsp; Prerequisites : 

```bash
- Install Python
- Install dependencies inside requirement.txt
- Start py app.py
- Open Browser on 127.0.0.1:3000
```

#### &nbsp; Limitations : 

```bash
- Models trained for only 4 batik motifs (Kawung, Megamendung, Merak Ngibing, Parang)
```
